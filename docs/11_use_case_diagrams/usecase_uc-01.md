# UC-01. Use Cases корпоративного пользователя

**Проект:** Корпоративная платформа унифицированных коммуникаций  
(**Unified Communications, UC**)

---

![Use Case диаграмма: Корпоративный пользователь](../assets/diagrams/usecase_uc-01.svg)

## Описание

Диаграмма Use Case для актора **Корпоративный пользователь** описывает функциональные возможности UC-платформы, доступные сотрудникам и агентам для ежедневной работы.

## Актор

**Корпоративный пользователь** — сотрудник, агент, использующий UC-платформу для коммуникаций в рамках операционной деятельности.

## Use Cases

### UC-Auth. Аутентификация и получение токена (SSO)
**Описание:** Аутентификация пользователя через корпоративный Identity Provider (OIDC/SAML 2.0) с поддержкой единого входа (SSO).  
**Связь с требованиями:** FR-01  
**Связь с sequence диаграммами:** UC-07

### UC-OutgoingCall. Исходящий голосовой вызов
**Описание:** Инициация исходящего голосового вызова через WebRTC-клиент с маршрутизацией на SIP/PSTN или внутренних абонентов.  
**Связь с требованиями:** FR-02, FR-12  
**Связь с sequence диаграммами:** UC-01

### UC-IncomingCall. Входящий вызов
**Описание:** Приём входящего вызова с поддержкой маршрутизации по правилам, очередям и группам.  
**Связь с требованиями:** FR-02, FR-11, FR-12  
**Связь с sequence диаграммами:** UC-02

### UC-Conference. Видеоконференция
**Описание:** Создание и участие в видеоконференциях с поддержкой аудио/видео, чата конференции и записи.  
**Связь с требованиями:** FR-03, FR-07  
**Связь с sequence диаграммами:** UC-04

### UC-Chat. Чат с сотрудниками
**Описание:** Обмен текстовыми и мультимедийными сообщениями в личных и групповых чатах с синхронизацией между устройствами.  
**Связь с требованиями:** FR-04, FR-12  
**Связь с sequence диаграммами:** UC-03

### UC-MultiDevice. Управление несколькими устройствами
**Описание:** Синхронизация состояния между несколькими устройствами пользователя (Web/Mobile/Desktop/SIP) для параллельных вызовов и единого опыта.  
**Связь с требованиями:** FR-12  
**Связь с sequence диаграммами:** UC-09

### UC-MessageHistory. Просмотр истории сообщений
**Описание:** Поиск и просмотр истории сообщений с фильтрацией и синхронизацией между устройствами.  
**Связь с требованиями:** FR-04, FR-12

### UC-Contacts. Управление контактами
**Описание:** Управление адресной книгой, контактами и группами с интеграцией корпоративных каталогов.  
**Связь с требованиями:** FR-01, FR-05

### UC-Presence. Просмотр статусов presence
**Описание:** Просмотр статусов доступности других пользователей (online/offline/busy/away) и подписка на обновления статусов.  
**Связь с требованиями:** FR-05  
**Связь с sequence диаграммами:** UC-03, UC-09

### UC-Recordings. Доступ к записям разговоров
**Описание:** Поиск, просмотр и скачивание записей разговоров с контролем прав доступа (RBAC/ABAC).  
**Связь с требованиями:** FR-07, FR-14  
**Связь с sequence диаграммами:** UC-08

### UC-CRMIntegration. Интеграция с CRM (click-to-call)
**Описание:** Инициация вызовов из CRM-системы (click-to-call) с автоматическим логированием взаимодействий.  
**Связь с требованиями:** FR-06, FR-08  
**Связь с sequence диаграммами:** UC-05

## Зависимости

- Все Use Cases, требующие аутентификации, **включают** UC-Auth
- UC-Conference **включает** UC-Chat (чат конференции)
- UC-Chat **включает** UC-Presence (статусы пользователей)
- UC-MessageHistory **расширяет** UC-Chat (дополнительная функциональность)

## Трассируемость

Диаграмма поддерживает архитектурные драйверы **AD-01**, **AD-02**, **AD-03** и решения из ADR-001, ADR-002, ADR-004.
